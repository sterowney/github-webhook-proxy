# github-webhook-proxy

```
npm install

node app.js

Github Webhook as (e.g. Slack Url): http://localhost:3000?webhook=[SOURCE_WEBHOOK_URL]

For local testing use ngrok: ./ngrok http 3000

This will proxy your service for local dev.

In practice, deploy your service somewhere e.g http://githubwebhookapp.myorg.dev
```
